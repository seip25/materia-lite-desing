Semantic HTML 🏗️

Semantic HTML consists of using elements that describe their meaning to both the browser and the developer. This improves accessibility, SEO, and code maintainability.

Basic Structure

structure.html 

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Page Title</title>
</head>
<body>
    <header>
        <nav>
            <!-- Navigation menu -->
        </nav>
    </header>
    
    <main>
        <section>
            <article>
                <!-- Main content -->
            </article>
        </section>
        
        <aside>
            <!-- Secondary content -->
        </aside>
    </main>
    
    <footer>
        <!-- Footer content -->
    </footer>
</body>
</html>

Key Semantic Elements

<header>

Represents the header of the page or a section. Typically contains the logo, title and main navigation.

<nav>

Contains the site's main navigation links. Should be used for the main menu, not for all link groups.

<main>

Contains the page's unique main content. There should only be one per document and it shouldn't include content that repeats on other pages.

<section>

Groups thematically related content. Each section should have its own heading (h2-h6).

<article>

Represents self-contained content that could be distributed independently (posts, news, articles).

<aside>

Contains related but non-essential content to the main content (sidebars, advertising, author biographies).

<footer>

Footer for the document or section. Typically contains copyright information, legal links, etc.

SEO Optimization 🔍

Improve your site's visibility in search engines with these recommended practices.

Essential Meta Tags

meta-tags.html 

<head>
    <!-- Character encoding -->
    <meta charset="UTF-8">
    
    <!-- Viewport for responsive design -->
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    
    <!-- Page title (max 60 chars) -->
    <title>Page Name - Brief description</title>
    
    <!-- Meta description (max 160 chars) -->
    <meta name="description" content="Detailed description of the page content">
    
    <!-- Keywords (optional) -->
    <meta name="keywords" content="word1, word2, word3">
    
    <!-- Author -->
    <meta name="author" content="Author Name">
    
    <!-- Favicon -->
    <link rel="icon" href="/favicon.ico" type="image/x-icon">
    
    <!-- Open Graph / Social Media -->
    <meta property="og:title" content="Title for social media">
    <meta property="og:description" content="Description for social media">
    <meta property="og:image" content="image-url.jpg">
    <meta property="og:url" content="https://yoursite.com/page">
    <meta property="og:type" content="website">
    
    <!-- Canonical URL -->
    <link rel="canonical" href="https://yoursite.com/page" />
</head>

SEO Best Practices

  • Semantic structure: Correctly use HTML5 elements like header, nav, main, section, article, aside and footer.
  • Heading hierarchy: Maintain a logical heading structure from h1 to h6 (only one h1 per page).
  • Alt attributes in images: Always describe images with alternative text.
  • Friendly URLs: Use descriptive, readable URLs (/products/sports-shoes instead of /p?id=123).
  • Quality content: Create original, useful and well-structured content.
  • Performance: Optimize loading time (image compression, CSS/JS minification).
  • Mobile-first: Design first for mobile and use responsive design.
  • Internal links: Create a good internal linking structure.
